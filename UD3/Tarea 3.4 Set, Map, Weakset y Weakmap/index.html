Nota: el código de esta tarea tiene su salida en la consola del navegador.

1. Indica qué es un Set y crea uno partiendo del array [1, 2, 3, 4, 4]. Justifica su tamaño

Un objeto Set es una colección de valores que no se repiten y que pueden ser de cualquier tipo. En el Set que hemos creado su tamaño es de 4 porque al tener el array dos valores repetidos uno de ellos se omite.

2. Justifica por qué funciona el encadenamiento en los métodos de set "new Set().add(11).add(22)"

Funciona porque los métodos devuelven el objeto con el que trabajan, de esa forma se pueden encadenar varias llamadas a los métodos

3. Indica cómo se devuelve un valor de un set

Un valor de un set es devuelto por referencia, y se puede iterar un Set según el orden de inserción.

4. Comparativa entre un array y un set: búsqueda de elementos, eliminación de elementos, búsqueda del elemento NaN, control de duplicados

La búsqueda de elementos se hace igual en ambos objetos, para eliminar un elemento en un Set
se debe usar el método delete pasándole como argumento el valor que se desea borrar, mientras que en el Array
se borra un elemento según su índice. En un Set un valor puede ser NaN, el cual aquí equivale a Undefined.
En el Set si un elemento repetido se intenta insertar este se omite, en un Array es necesario iterar
cada uno de sus elementos para comprobar si el elemento a insertar ya existe.

5. Crea un set donde se almacenen los alumnos ausentes. Repítelos y muéstralos.

Hemos creado un Set con los alumnos Pepe, María, Sara, María. Como María estaba repetido no la ha añadido por segunda vez

6. Indica qué es un Map y crea uno partiendo del array. Partiendo de él, crea tres arrays: Uno con las claves, otro con los valores y otro con los objetos.

El objeto Map almacena pares de clave-valor, cualquier valor u objeto puede ser usado como clave o como valor.
